Data Flow — Filter Selection to Rendered Data

1

User changes a filter dropdown

FilterBar.vue binds each <select> via v-model to a ref inside the useFilters() composable. No event emits — the ref mutation is the signal.

selectedLocation = 'Tokyo'
2

View watcher fires

Each view (e.g. Orders.vue) has a watch() on the relevant filter refs. When any watched ref changes, it calls the view's async loadData() function.

watch([selectedPeriod, selectedLocation, selectedCategory, selectedStatus], loadOrders)
3

API client builds query string

api.js calls getCurrentFilters(), then appends each non-"all" / non-null value to a URLSearchParams object.

GET /api/orders?warehouse=Tokyo&status=shipped
4

FastAPI route handles request

The route receives typed query params (Optional[str] = None). It chains apply_filters() for warehouse/category/status, then filter_by_month() for date range. Both functions skip filtering when the param is None or "all".

apply_filters(orders, warehouse='Tokyo', status='shipped')
5

Pydantic validates and serializes

FastAPI uses the response_model=List[Order] annotation to validate each item against the Pydantic schema before sending. Optional fields are excluded when null.

response_model=List[Order]
6

Vue updates reactive data & re-renders

The JSON array is assigned to a reactive ref (e.g. orders). Computed properties like filteredItems re-run automatically. The template re-renders the updated table.

orders.value = response.data

Frontend Composables (Global State)

🔍
useFilters() — Global filter state
Singleton composable holding the 4 shared filter refs. All views read from and watch these refs. FilterBar writes to them. No event bus or store needed — mutation of the shared refs propagates reactively.
selectedPeriod selectedLocation selectedCategory selectedStatus getCurrentFilters() resetFilters() hasActiveFilters
🌐
useI18n() — Internationalization + currency
Manages locale (EN / JA) stored in localStorage. Provides a t() lookup function with English fallback. Derives the active currency (USD vs JPY) from the current locale.
locale t(key) currentCurrency setLocale()
👤
useAuth() — Mock authentication
Provides a mock logged-in user with language-aware tasks. Exposes a computed currentUser object, auth state, and a logout handler. No real auth — demo only.
currentUser isAuthenticated logout() getInitials()
